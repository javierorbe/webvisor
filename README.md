# WebVisor

A TypeScript framework for building 3D interactive apps with WebGL.

## Run example app

Install node dependencies:

```
npm install
```

Build the project:

```
npm run build
```

Run a web server (from the root folder) using, for example, [http-server](https://www.npmjs.com/package/http-server):

```
http-server
```
